Langkah - langkah penggunaan aplikasi :
1. download project laravel disitu terdapat database beserta code apinya
2. kemudian lakukan config dengan database masing-masing
3. jalankan perintah "php artisan serve --host=(ip masing-masing) --port=8000(boleh bebas portnya)
4. ke project flutternya
5. sesuaikan dulu url api dengan alamat ipnya dengan ip laptop masing-masing
6. jalankan projectnya dengan perintah "flutter run" atau didebugging difitur visual studio code/android studio masing-masing
7. login dengan email & password yang tertera di folder tempat anda menyimpan kemudian di folder "lib\app\modules\login\views"
